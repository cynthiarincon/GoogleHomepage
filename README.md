# Google Homepage

A recreation of Google's homepage built as part of GitHub Challenge #1.

## Project Overview

This project replicates the look and feel of Google's homepage using HTML, CSS, and JavaScript.

## Tech Stack

- HTML
- CSS
- JavaScript

## Installation

1. Clone the repository
2. Open `index.html` in your browser
